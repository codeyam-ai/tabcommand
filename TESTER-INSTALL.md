# TabCommand — Tester Install (v0.13)

1. Unzip `tabcommand-v0.13-unpacked.zip`, then open `chrome://extensions`, toggle **Developer mode** on (top-right), and click **Load unpacked** — select the unzipped folder (the one with `manifest.json` directly inside it).
2. Heads-up: this build does **NOT** auto-update. Each time you get a new zip, remove the old TabCommand from `chrome://extensions` and Load unpacked again.
